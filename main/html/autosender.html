AutoSender

O Auto Sender WhatsApp é um script em Python desenvolvido com o objetivo de automatizar o envio de mensagens pelo WhatsApp Web. O projeto utiliza a biblioteca pywhatkit para acessar o WhatsApp Web e enviar mensagens de forma programada para contatos específicos.

Esse projeto foi importante para entender melhor como o Python pode ser usado para automação de tarefas. Também foi uma das minhas primeiras experiências com bibliotecas que interagem com interfaces gráficas, como o tkinter. Foi um ótimo exercício de lógica e controle de tempo no código.
