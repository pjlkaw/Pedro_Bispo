PokeSearch

O PokeSearch é uma aplicação web desenvolvida com foco em praticar o consumo de APIs públicas, utilizando a API do site PokeAPI. O projeto utiliza a PokéAPI para exibir informações detalhadas sobre os Pokémon pesquisados, como nome, tipo, habilidades e imagem.

O projeto foi de grander importâcia para conhecer mais sobre o JavaScript, foi a minha primeira experiência com uma API. Além disso foi o projeto que fez eu entender como funciona Git e GitHub.
