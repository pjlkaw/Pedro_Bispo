Decode

O DeCode é uma aplicação web criada com o objetivo de praticar lógica de programação e manipulação do DOM com JavaScript. O projeto simula diferentes tipos de codificações de texto, como Base64, Código Alfabético, Cifra de César, Código Morse, Hexadecimal e ROT13, permitindo ao usuário experimentar e entender como cada método transforma a informação.

Esse projeto foi muito importante para fixar os conhecimentos que eu vinha adquirindo em estudos anteriores. Foi nele que consegui aplicar com mais segurança conceitos como eventos, funções, estruturas de repetição e condicionais. Além disso, me ajudou a organizar melhor o uso do HTML e CSS em conjunto com o JavaScript, reforçando minha base no desenvolvimento front-end.
