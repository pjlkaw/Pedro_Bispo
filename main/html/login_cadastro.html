Login & Signup

Este projeto é uma aplicação web simples que implementa funcionalidades de cadastro e login de usuários. Desenvolvido com foco no aprendizado de back-end, utiliza Node.js para criar um servidor que gerencia as requisições de cadastro e autenticação.

O sistema armazena os dados dos usuários em um arquivo JSON, permitindo a persistência das informações sem a necessidade de um banco de dados complexo. As páginas de cadastro e login são construídas com HTML, CSS e JavaScript, proporcionando uma interface básica para interação com o usuário.

Este projeto foi fundamental para compreender os conceitos de rotas em servidores, manipulação de arquivos no back-end e a lógica envolvida na autenticação de usuários. Além disso, serviu como introdução prática ao uso do Node.js e ao desenvolvimento de aplicações web com persistência de dados.
